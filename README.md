# 🛍️ Nostra E-Commerce Website

Nostra is a fully responsive e-commerce clothing website developed using **HTML, CSS, and JavaScript**.  
This project is designed to provide a modern and smooth online shopping experience with interactive features like **search functionality** and **product filtering**.

## 🚀 Features

- Fully responsive design for mobile, tablet, and desktop
- Product search functionality using JavaScript
- Product filter functionality
- Interactive UI design
- Modern clothing brand website layout
- Clean and user-friendly interface
- Smooth navigation between pages

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## 💡 Functionalities Implemented

### 🔍 Search Functionality
Users can search products by entering keywords, and matching products will be displayed dynamically using JavaScript.

### 🎯 Filter Functionality
Users can filter products based on categories and preferences for easy product selection.

### 📱 Responsive Design
The website is fully responsive and works perfectly across all screen sizes.

## 📂 Project Purpose
This project was developed to improve frontend development skills and practice real-world e-commerce website functionality using JavaScript.
